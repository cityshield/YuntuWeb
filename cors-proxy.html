解决方案

如果出现 "Failed to fetch" 错误：

  1. 检查 API 服务状态 - 确保后端服务正在运行
  2. 检查网络连接 - 确保可以访问 API 服务器
  3. 检查 CORS 配置 - 后端需要允许跨域请求
  4. 使用代理服务器 - 通过同域代理转发请求

后端 CORS 配置示例（Flask）：

from flask_cors import CORS

app = Flask(__name__)
CORS(app, origins=['*'])  # 允许所有来源
