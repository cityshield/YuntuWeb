# Claude 项目指令

## 语言偏好
- **请始终使用中文回答所有问题和对话**
- 所有的解释、说明、代码注释都使用中文
- 保持专业、友好和高效的沟通风格

## 项目信息
这是一个名为"盛世云图"（YuntuCV）的Web应用项目，专注于为影视动漫行业提供AI技术解决方案。


## 技术栈
- 前端：HTML5, CSS3, JavaScript (原生)
- 后端API：Python (通过api-config.js配置)
- 主题：支持亮色/暗色模式
- 环境：支持本地开发和生产环境切换

## 代码规范
- 使用中文注释
- 保持代码简洁清晰
- 遵循现有的代码风格
- 能不创建新的文件，就不要创建新的文件，增加代码的可复用性
- 所有的新增功能，都需要记录在日志里面且用中文，并且输出到控制台，方便调试
- 功能测试完成之后，要把用于测试的文件给删掉
